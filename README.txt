Como usar:
1. Abra o arquivo index.html no navegador.
2. Para ativar o WhatsApp, abra js/script.js.
3. Substitua:
   const numero = "SEU_NUMERO_AQUI";
   por seu número no formato internacional, exemplo:
   const numero = "5511999999999";
